#Atención
Selecciona un código de la lista para leer más sobre él.

Para ejecutarlos, es necesario compilarlos, desde línea de comandos se hace utilizando la siguiente sintaxis:
```
g++ codigo.cpp -o nombreEjecutable
```

Donde `codigo.cpp` representa el código que quieren compilar, y `nombreEjecutable` es como quieren que se llame el ejecutable.

En caso de utilizar [Eclipse](http://www.eclipse.org/downloads/), solo es cuestión de importar el archivo a un nuevo proyecto, compilarlo y ejecutarlo desde ahí.
